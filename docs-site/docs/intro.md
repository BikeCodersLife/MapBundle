# MapBundle Documentation

Symfony bundle for managing self-hosted vector map tiles using Protomaps

## Key Features

- PMTiles single-file archives
- S3 integration
- Cost-effective (€0 vs €5+/1k tiles)
- MapLibre GL JS integration
- Symfony 6.4+/7.0 compatible
